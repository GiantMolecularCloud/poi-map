get_new_poi_modal() → dash_bootstrap_components.Modal

Get a modal to add a new POI.

The modal contains the following fields: - Location: Display the coordinates of the clicked location. - Title: Input field for the title of the POI. - Category: Dropdown to select the category of the POI. - Date: Date picker to select the date of the POI. - Description: Input field for the description of the POI.

Additionally, a header and footer with a button to create the POI is included.

Returns:

Modal.
